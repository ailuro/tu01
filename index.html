Theme Info

This theme was inspired by Miraculous Ladybug and its fandom, and therefore this theme is dedicated to that particular fandom.

This is my first public theme, so it’s probably not perfect! I’m mainly doing this because I enjoy it, so excuse any mistakes, I will try my best to fix them if I can.

Any questions should be directed to my theme blog.

The default colours are as shown on this demo blog, and you are perfectly welcome to leave them as they are if you really want to. The sidebar looks best if you use a square image.
